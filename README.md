## How to Play

1. Open `index.html` in your web browser.  
2. Click **Start** to begin the game.  
3. Choose **Rock**, **Paper**, or **Scissors**.  
4. See the result: Win, Lose, or Draw!  
5. Enjoy the game and restart anytime.
